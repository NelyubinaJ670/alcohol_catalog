# alcohol_catalog

# Каталог поставщиков и покупателей

### Описание проекта:
Проект, где поставщики алкогольной продукции могут находить себе клиентов, а клиенты поставщиков.

## Технологии
- Python 3.10
- Django 5.1.7
- djangorestframework 3.15.2
